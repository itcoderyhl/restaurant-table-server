# restaurant-table-server
基于SpringBoot+Vue前后端分离的餐厅订座点餐管理系统，系统包含管理员、用户角色，功能如下： 管理员：首页统计（包间统计、7天订餐金额统计）；用户管理；楼层管理；包间管理；菜单管理；包间预约；订单管理；留言管理；系统公告；个人信息；密码修改。 用户：注册、登录系统；包间查看（预约）；菜单查看；包间预约（取消预约、入座、离座归还）；订单管理（点菜中、制作中、已出餐、已完成、已取消）；留言管理；公告查看；个人资料；修改密码。 系统界面美观大方，功能丰富，使用了SpringBoot、Vue、ElementUI、Echarts等技术栈，有需要可提供指导。
